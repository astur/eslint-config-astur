# eslint-config-astur

[eslint](https://eslint.org) config that I like to use

[![NPM version][npm-image]][npm-url]

## Install

```bash
npm install eslint eslint-config-astur
```

## Usage

Make `eslint.config.js` like this:

```js
export {default} from 'eslint-config-astur';
```
